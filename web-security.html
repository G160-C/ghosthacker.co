Introduction to Web Security

Web Security is the practice of protecting websites and web applications from attackers. This module will help you understand how the internet works, what vulnerabilities exist in web applications, and how to defend against them.

Lesson 1: HTTP Basics

  • Understand the HTTP request/response cycle
  • Learn about methods: GET, POST, PUT, DELETE
  • Status codes and headers

Lesson 2: Common Vulnerabilities

  • Cross-Site Scripting (XSS)
  • SQL Injection (SQLi)
  • Cross-Site Request Forgery (CSRF)
  • Directory Traversal
  • File Upload Vulnerabilities

Lesson 3: Hands-on Labs

  • TryHackMe: Web Fundamentals
  • PortSwigger Labs (Burp Academy)
  • HackTheBox: Web Challenges

Lesson 4: Prevention & Mitigation

  • Input validation and sanitization
  • Using prepared statements
  • Security headers (CSP, HSTS, etc.)
  • Rate limiting and logging

Lesson 5: Tools You’ll Use

  • Burp Suite (Community)
  • OWASP ZAP
  • Postman
  • Browser DevTools
