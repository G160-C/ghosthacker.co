1. Why Linux for Hackers?

Most hacking tools are written for Linux. Kali Linux, Parrot OS, and BlackArch are hacker-focused distros. Linux provides flexibility, open-source tooling, and a strong command-line interface.

2. Terminal Basics

Learn these basic Linux commands: 


pwd        # Show current directory
ls         # List files
cd         # Change directory
mkdir      # Create a folder
rm         # Remove file/folder
man ls     # Manual for a command

3. File & User Permissions

Understand `chmod`, `chown`, and permission flags:


chmod +x script.sh     # Make file executable
chown user:group file  # Change ownership
ls -l                  # See file permissions

Permission flags: `r` = read, `w` = write, `x` = execute

4. Networking Commands

Essential networking tools:


ifconfig / ip a        # View IP addresses
ping google.com        # Check connectivity
netstat -tuln          # List open ports
nmap 192.168.1.1       # Scan for services

5. Bash Scripting Essentials

Automate tasks with bash scripts: 


#!/bin/bash
for ip in {1..254}; do
  ping -c 1 192.168.1.$ip &> /dev/null && echo "Host 192.168.1.$ip is up"
done

Save this as `scan.sh`, then run: chmod +x scan.sh && ./scan.sh
